# GRE Analytical Writing 완전 정리

## 개요

- **두 과제**: Analyze an Issue (30분) + Analyze an Argument (30분)
- **점수**: 0~6점 (0.5점 단위), 두 과제 평균
- **채점 기준 핵심**: 비판적 사고 + 분석적 글쓰기 능력

---

## Analyze an Issue Task

### 과제 설명
- 광범위한 관심사의 이슈에 대해 입장을 수립하고, 특정 지시문에 따라 논거를 전개
- 정답 없음 — 특정 지시문을 얼마나 잘 따르고 논증하는지 평가

### 6가지 지시문 유형 (Issue Instruction Types)

| # | 핵심 요구 |
|---|---|
| 1 | 동의/비동의 범위 논의, 성립하는 경우와 그렇지 않은 경우 고려 |
| 2 | 권고사항 채택이 유리한 경우와 불리한 경우 구체적으로 서술 |
| 3 | 자신의 입장에 도전하는 가장 강력한 반론도 다룰 것 |
| 4 | 두 관점 중 자신의 입장에 더 가까운 쪽 선택, 두 관점 모두 논의 |
| 5 | Claim과 그 근거(reason) 모두에 동의/비동의 범위 논의 |
| 6 | 정책 시행의 가능한 결과를 고려하고 그것이 입장에 미치는 영향 설명 |

### 채점 기준 (Issue)

| 점수 | 특징 |
|---|---|
| **6** | 통찰력 있는 입장 + 설득력 있는 이유/예시 + 논리적 구성 + 유창하고 정확한 표현 |
| **5** | 사려 깊은 입장 + 논리적 이유/예시 + 잘 조직됨 + 명확한 표현 |
| **4** | 명확한 입장 + 관련 이유/예시 + 적절히 구성됨 + 충분한 언어 통제 |
| **3** | 일부 역량 있음: 모호한 입장, 약한 지지, 제한된 구성, 명확성 문제 중 1+ |
| **2** | 심각한 약점: 불명확한 입장, 근거 거의 없음, 불량 구성, 빈번한 오류 |
| **1** | 근본적 결핍: 이해 없음, 체계 없음, 심각한 언어 문제 |
| **0** | 주제 이탈, 외국어, 과제 복사, 판독 불가 |

### 고득점 전략
1. **지시문 유형 먼저 파악** — 6가지 중 어떤 것인지 확인
2. **조건부 입장 가능** — 완전 동의/비동의보다 "~한 경우에는 동의" 형태가 복잡성을 보여줌
3. **구체적 예시 필수** — 역사적·과학적·개인적 예시 모두 가능, 충분히 발전시킬 것
4. **반론 인식** — 자신의 입장에 도전하는 근거를 인식하고 방어하거나 수용
5. **30분 배분**: 브레인스토밍(5분) → 작성(22분) → 검토(3분)

## Analyze an Argument Task

### 과제 설명
- 짧은 논거 지문을 읽고 **논리적 건전성**을 비판적으로 검토
- 지문 필자의 주장에 동의/비동의하는 것이 **아님**
- 논거의 증거, 가정, 결론, 추론의 타당성을 분석

### 읽을 때 주의할 4가지 요소
1. 증거·지지·증명으로 제시된 것
2. 명시적으로 주장하거나 결론 내린 것
3. 정당화 없이 가정된 것
4. 명시되지 않았으나 반드시 따라오는 것

### 8가지 지시문 유형 (Argument Instruction Types)

| # | 핵심 요구 |
|---|---|
| 1 | 논거 평가에 필요한 구체적 증거 논의 |
| 2 | 명시적·암묵적 가정 검토, 가정이 틀렸을 때의 함의 설명 |
| 3 | 권고사항과 그 근거가 합리적인지 판단하기 위해 답해야 할 질문 논의 |
| 4 | 조언과 그 근거가 합리적인지 판단하기 위해 답해야 할 질문 논의 |
| 5 | 권고사항이 예측 결과를 낳을 가능성 판단에 필요한 질문 논의 |
| 6 | 예측과 그 근거가 합리적인지 판단하기 위해 답해야 할 질문 논의 |
| 7 | 제안된 설명에 대한 대안적 설명 제시 |
| 8 | 결론과 그 근거가 합리적인지 판단하기 위해 답해야 할 질문 논의 |

### 핵심 개념 용어

| 용어 | 정의 |
|---|---|
| alternative explanation | 관찰된 사실을 설명할 수 있는 경쟁적 설명 |
| assumption | 진술을 유지하기 위해 반드시 참이어야 하는, 종종 명시되지 않은 믿음 |
| conclusion | 추론 과정의 끝점 |
| counterexample | 주장을 반박하는 실제 또는 가상의 예 |
| evaluation | 논거의 증거 및 이유의 질에 대한 평가 |

### 고득점 전략
1. **자신의 의견 표명 금지** — 논거의 논리적 건전성만 평가
2. **가정들을 연결하여 논의** — 고립적 나열이 아닌 논리적 순서 속에서 검토
3. **대안적 설명 제시** — 가정이 틀렸을 때 어떤 다른 설명이 가능한지
4. **수치·통계 비판적 해석** — 수치 자체보다 결론을 지지하는지 여부 평가
5. **전환어로 논지 흐름 유지**

## 점수 수준별 설명 (Score Level Descriptions)

| 점수 | 설명 |
|---|---|
| 6/5.5 | 복잡한 아이디어에 대한 통찰력 있는 심층 분석; 설득력 있는 이유·예시; 논리적 구성; 다양한 문장·정확한 어휘 |
| 5/4.5 | 전반적으로 사려 깊은 분석; 논리적 이유·잘 선택된 예시; 잘 조직됨; 명확한 표현 |
| 4/3.5 | 적절한 분석; 관련 이유·예시; 합리적 명확성으로 표현; 일부 오류 |
| 3/2.5 | 일부 역량 있음: 발전 부족, 구성 약함, 명확성 문제 중 하나 이상 |
| 2/1.5 | 심각한 약점: 분석 부족, 구성 불량, 오류가 의미를 모호하게 함 |
| 1/0.5 | 근본적 결핍: 과제와 무관하거나 혼란스러운 내용, 발전 거의 없음 |
| 0 | 과제 미이행, 복사, 외국어, 판독 불가 |

## 샘플 에세이 고득점 특징 요약

### Issue Task 6점 에세이 특징
- 명확하고 통찰력 있는 입장 수립
- 구체적·충분히 발전된 예시 활용
- 단락 간 명확한 전환어 사용
- 다양하고 복잡한 문장 구조

### Argument Task 6점 에세이 특징
- 핵심 가정 3~4개를 파악하고 각각 철저히 검토
- 가정들 간의 논리적 연결 관계 고려
- 효과적인 단어 선택 ("rife with assumptions")
- 각 가정이 틀렸을 때의 함의를 구체적으로 제시
