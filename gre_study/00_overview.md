# GRE General Test 개요

## 시험 구조 (컴퓨터 시험 기준)

| 섹션 | 문항 수 | 시간 |
|---|---|---|
| Analytical Writing (2개 과제) | Issue 1개 + Argument 1개 | 각 30분 |
| Verbal Reasoning (2섹션) | 섹션당 20문항 | 섹션당 30분 |
| Quantitative Reasoning (2섹션) | 섹션당 20문항 | 섹션당 35분 |
| Unscored (위치 무작위) | 20문항 | 가변 |

- **총 시험 시간**: 3시간 45분 (비채점 섹션 포함)
- **Analytical Writing 섹션이 항상 첫 번째**
- 나머지 섹션 순서는 무작위
- **섹션 적응형(section-level adaptive)**: Verbal·Quant 2번째 섹션은 1번째 성과에 따라 난이도 결정

## 점수 체계

### Analytical Writing
- 0~6점 척도 (0.5점 단위)
- 두 과제 점수 평균 → 하나의 점수로 보고
- 채점: 훈련된 채점자 1인 + ETS e-rater(AI) → 불일치 시 두 번째 인간 채점자 추가

### Verbal & Quantitative Reasoning
- 맞힌 문항 수로 결정 (오답 감점 없음)
- **전략: 모든 문항에 반드시 답하기** (추측이라도)
- 각 문항의 배점은 동일 → 어려운 문항에 시간 낭비 금지

## 시험 응시 전략

### 컴퓨터 시험
- Mark & Review 기능: 문항을 표시하고 나중에 다시 풀기 가능
- 섹션 내 자유롭게 앞뒤 이동 가능
- 섹션이 끝나면 이전 섹션으로 돌아갈 수 없음
- 스크래치 페이퍼 제공 (필요 시 추가 요청 가능)
- 온스크린 계산기 사용 가능 (Quantitative 섹션)

### 시간 관리
- Verbal: 30분/20문항 → 문항당 평균 1.5분
- Quant: 35분/20문항 → 문항당 평균 1.75분
- 10분 휴식: 3번째 섹션 후
- 1분 휴식: 나머지 섹션 사이
