# GRE Verbal Reasoning 완전 정리

## 개요
- 2섹션 × 20문항, 섹션당 30분
- 전체 문항의 약 절반 = 독해(RC), 절반 = 빈칸완성(TC·SE)
- 대학원 수준의 산문을 읽고 이해하는 광범위한 능력 평가

---

## 문제 유형 1: Reading Comprehension (독해)

### 평가하는 능력
- 개별 단어·문장·단락의 의미 이해
- 중요 포인트와 부수적 포인트 구별
- 지문에서 결론 도출 및 추론
- 저자의 관점(perspective)과 가정(assumptions) 파악
- 주장의 강점과 약점 파악
- 대안적 설명 고려

### 지문 구성
- 시험 전체에 약 **10개 지문**, 주로 **1단락** 길이
- 주제 영역: 인문학 / 사회과학(비즈니스 포함) / 자연과학 균형 배분
- 지문당 질문 수: 1~6개

### RC 세 가지 문항 유형

**유형 1: Select One Answer Choice (단일 정답 5지선다)**
- 전략:
  - 정답을 알더라도 모든 선택지 확인
  - 부분적으로만 맞는 선택지 주의
  - 단순히 "사실인 진술"이라는 이유로 선택 금지
  - 단어 의미 문제는 문맥상 사용 방식에 주의

**유형 2: Select One or More Answer Choices (복수 정답)**
- 3개 선택지, 해당하는 것 모두 선택 (1~3개 가능)
- 선택지 옆 **사각형(□)** 표시
- **부분 정답 없음** — 정답 전부를 골라야만 점수 인정
- 전략:
  - 각 선택지를 독립적으로 평가
  - 3개 모두 정답일 수도 있음

**유형 3: Select-in-Passage (지문 내 문장 선택)**
- 특정 설명에 해당하는 문장을 지문에서 직접 클릭
- **지필 시험에는 등장하지 않음** (대신 유사 객관식으로 대체)
- 전략:
  - 각 문장을 개별적으로 평가
  - 설명이 부분적으로만 적용되면 선택 금지

### RC 핵심 전략
1. **능동적 독해** — 질문하고, 가설을 세우고, 평가하면서 읽기
2. **외부 지식 의존 금지** — 지문 내 정보만으로 답하기
3. **주요 아이디어 vs 부수적 아이디어 구별**
4. **저자가 주장하는 것 vs 단순히 보고하는 것 구별**
5. **아이디어 간 관계 파악**: 대조 / 일관성 / 뒷받침 / 상술 / 적용
6. **어려운 지문은 나중에** — 모르는 주제라도 지문 내에서만 답 가능

## 문제 유형 2: Text Completion (빈칸 완성)

### 형식
- 1~5개 문장 지문 + 빈칸 1~3개
- 빈칸당 3개 선택지 (빈칸이 1개인 경우 5개 선택지)
- 서로 다른 빈칸의 선택지는 독립적
- **부분 정답 없음**

### 전략
1. 모든 조합을 시도하지 마라 (시간 낭비)
2. 지문 전체를 훑어 전반적 의미 파악
3. **신호어 주목**: although, moreover, however, since, so that 등
4. **자신의 단어를 먼저 생각** → 그 뒤 선택지에서 유사한 것 탐색
5. **더 쉬운 빈칸부터** 채우고 그 결과로 나머지 결정
6. 답 선택 후 논리·문법·문체 일관성 검증

### 예시 풀이
> "Vain and prone to violence, Caravaggio could not handle success: the more his (i)___ as an artist increased, the more (ii)___ his life became."

- 콜론 이후가 콜론 이전을 설명 → 성공의 결과로 (i)는 명성 증가
- 명성이 높아질수록 삶이 어떻게 됐는가 → (ii)는 격동적
- **정답**: (i) eminence, (ii) tumultuous

## 문제 유형 3: Sentence Equivalence (문장 등가)

### 형식
- 단일 문장 + 빈칸 1개
- **6개 선택지 중 2개 선택**
- 두 선택지 각각이 일관된 문장을 만들면서 **두 문장의 의미가 같아야 함**
- 선택지 옆 **사각형(□)** 표시
- **부분 정답 없음**

### 핵심 함정
> "단순히 같은 의미의 두 단어를 찾으려 하지 마라"

- 같은 의미 단어 쌍이라도 문장에 논리적으로 맞지 않으면 오답
- 정답인 두 단어가 정확히 같은 의미가 아닐 수도 있음
- 중요한 것: **완성된 두 문장이 같은 의미**인 것

### 전략
1. 문장 전체를 읽어 전반적 의미 파악
2. 신호어(although, moreover 등) 주목
3. **먼저 자신만의 단어를 생각** → 유사한 단어 쌍 탐색
4. 하나만 있다면 고집하지 말고 다른 해석 시도
5. **두 선택지 모두 검증** — 논리·문법·문체 일관성 + 두 문장의 의미 동일성

### 예시 풀이
> "Although it does contain some pioneering ideas, one would hardly characterize the work as ___."

- "Although" → 일부 개척적 아이디어가 있지만 전체는 그렇지 않음
- 빈칸 = "개척적이지 않다"의 의미를 뒤집는 단어
- original / innovative → 문장이 논리적이고 두 문장의 의미 동일
- **정답**: original, innovative

## 전체 Verbal 전략 요약

| 원칙 | 내용 |
|---|---|
| 능동적 독해 | 수동적 읽기 금지; 질문하고 평가하면서 읽기 |
| 문맥 우선 | 외부 지식 배제; 지문/지문 맥락 내에서만 답하기 |
| 먼저 자신의 답 | TC·SE에서 선택지 보기 전에 스스로 답을 생각 |
| 신호어 포착 | although, moreover, however, since, so that 등 |
| 빈칸 순서 유연성 | TC에서 첫 번째 빈칸부터 채울 필요 없음 |
| 부분 정답 없음 | 복수 정답 유형(RC·SE·TC)은 전부 맞아야 점수 |
| 어려운 지문 나중에 | 생소한 RC 지문은 스킵 후 나중에 풀기 |
| 최종 검증 | 모든 유형에서 선택 후 논리·문법·문체 일관성 확인 |
